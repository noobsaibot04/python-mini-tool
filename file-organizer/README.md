# File Organizer

Simple Python automation tool that organizes files into folders automatically.

## Features
- Organizes image files
- Organizes video files
- Organizes music files
- Organizes document files
- Automatically creates folders
- User can choose folder path

## Supported File Types

### Images
- .jpg
- .jpeg
- .png

### Videos
- .mp4
- .mkv
- .avi

### Music
- .mp3
- .wav

### Documents
- .pdf
- .docx
- .txt

## Technologies
- Python
- os module
- shutil module

## Example
```bash
Enter folder path:
/home/kali/Downloads
```

The program automatically creates:
- Images
- Videos
- Music
- Documents

folders and moves files into them.
