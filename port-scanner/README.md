# Port Scanner

Simple Python port scanner using the socket module.

## Features
- Scans a specific port
- Checks if a port is open or closed
- User can enter target IP or website
- Uses Python socket programming

## Technologies
- Python
- socket module

## Example
```bash
Enter target IP or website: google.com
Enter port: 80

Port 80 is OPEN
```

## How It Works
The program tries to connect to the target port.

- If the connection is successful:
  - Port is OPEN

- If the connection fails:
  - Port is CLOSED
